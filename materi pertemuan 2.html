1. Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup untuk membuat struktur dasar halaman web. HTML mengatur isi & susunan konten, bukan tampilan. HTML = struktur | CSS = desain | JS = interaksi

2. Struktur Dasar Dokumen HTML

<!DOCTYPE html>
<html>
<head>
    <title>Judul</title>
</head>
<body>
    Halo Dunia!
</body>
</html>
