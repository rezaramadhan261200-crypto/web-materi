1. Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa untuk mengatur tampilan, warna, ukuran, posisi, dan gaya elemen pada halaman web. HTML = struktur CSS = tampilan (styling)

2. Contoh Selektor CSS

h1 {
    color: blue;
    text-align: center;
}

3. Ringkasan

CSS digunakan untuk mengatur tampilan halaman web. Cara terbaik menggunakan CSS adalah external, karena rapih dan mudah dipelihara. Styling utama: color, font, alignment, border, background, margin, padding.
