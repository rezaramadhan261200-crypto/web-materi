Apa itu Web?

Web adalah sistem informasi yang saling terhubung menggunakan protokol HTTP.

Cara Kerja Web

  • Pengguna mengetik URL ke browser
  • Browser mengirim request ke server
  • Server mengembalikan file HTML/CSS/JS
  • Browser menampilkan halaman

1. Pengertian Web

Web (World Wide Web) adalah sistem yang memungkinkan pengguna mengakses informasi melalui internet menggunakan browser dan protokol HTTP/HTTPS. Internet = jaringan | Web = layanan di atas internet

2. Sejarah Singkat Web

  • 1989 — Tim Berners-Lee mengenalkan WWW
  • 1991 — Website pertama dirilis
  • 2004 — sekarang — Web berkembang menjadi interaktif (Web 2.0 & seterusnya)

3. Istilah Penting

  • Domain: nama website (google.com)
  • Hosting: tempat menyimpan file web
  • URL: alamat halaman web
  • HTTP/HTTPS: protokol komunikasi
  • Web Page: halaman tunggal di web

4. Komponen Web

  • Client/Browser: mengirim permintaan & menampilkan halaman
  • Server/Web Server: memproses permintaan dan memberikan respons
  • Database (opsional): menyimpan data pada web dinamis

5 . Cara Kerja Web (Alur Singkat)

    1. Pengguna mengetik URL di browser

    2. Browser mengirim request ke server

    3. Server memproses & mengirim response (HTML, CSS, JS, data)

    4.Browser menampilkan halaman ke pengguna
